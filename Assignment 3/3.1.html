10. Write advantage and limitations of plots created using R base packages based on these nine steps

Advantages - - The R base plot is already available without loading any package so we can use it easily. - The plots are customizable i.e. we can add lable, topics, ablines, etc. - It supports multiple types of graph such as boxplot, barplot, timeseries plot, scatter plot, etc.

Limitations - - It takes time to plot for variables containing large numbers of rows - It doesn’t supports complex graph which can be generated using ggplot2
